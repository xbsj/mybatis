什么是Mybatis?
      mybatis是一款优秀的持久层框架，它支持定制SQL、存储过程及高级映射。MyBatis避免了几乎所有的JDBC代码和手动设置参数以及获取结果集。
MyBatis可以使用简单的XML或注解来配置和映射原生信息，将接口和Java的POJO(普通的Java对象)映射成数据库中的记录。
